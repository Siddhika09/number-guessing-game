# number-guessing-game
Flask number guessing game with modern UI
🎯 Number Guessing Game

A fun and interactive Number Guessing Game built using Flask (Python) with a modern UI.

---

🚀 Features

- 🎮 Interactive gameplay
- 🔢 Guess numbers between 1–50
- 📉 Hints (Too High / Too Low)
- 📊 Attempt counter
- 📜 Guess history
- 🎉 Winning screen

---

🛠️ Tech Stack

- Python
- Flask
- HTML, CSS, JavaScript

---

▶️ How to Run Locally

pip install -r requirements.txt
python app.py

Then open:

http://127.0.0.1:5000/
